# 🧩 Sudoku Game

## 📌 Overview

This project is an enhanced version of a previously developed Sudoku game, redesigned with a modern user interface and improved user experience. It introduces a time-based constraint to make gameplay more engaging and challenging.

## 🚀 Features

* 🎨 Redesigned and user-friendly UI
* ⏱️ Time constraint for added challenge
* 🧠 Interactive Sudoku gameplay
* ✅ Input validation for correct moves
* 📱 Responsive design

## 🎯 Objective

The aim of this project is to provide an engaging Sudoku experience by combining intuitive design with time-based gameplay, helping users improve both speed and accuracy.

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```
Sudoku-Game/

```

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start playing 🎉
